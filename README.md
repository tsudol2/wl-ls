# CS474 Project 4 - Weird Language Language System in C++
## Tyler Sudol
To build the project, execute the scripts:

    ./configure.sh
    ./build.sh
    ./run.sh

Or manually run commands from root directory:

    cmake -S . -B build/
    cd build
    make && make install
    cd ..
    ./wl-ls
    ./wl-gtest
